### Детальное сравнение с указанием конкретных URL

#### 1. Tencent Cloud (腾讯云)

- URL:
  - Продуктовая страница: https://cloud.tencent.com/product/tmt
  - Документация API: https://trtc.io/document/66148
  - Блог с демо: https://www.tencentcloud.com/dynamic/blogs/sample-article/100840
- Тип: Полный цикл speech-to-speech (STT → перевод → TTS)
- Задержка: Миллисекундная (<500ms)
- Поддержка русского: ✅ Да (в списке 15 языков)
- Бесплатно: ✅ 10,000 символов/месяц для новых пользователей
- Языки: 15 (китайский, английский, испанский, португальский, французский, немецкий, русский, арабский, японский, корейский, и др.)
- Особенности:
  - AI-шумоподавление и распознавание голосовых отпечатков
  - Контекстно-зависимое ASR для специализированной терминологии
  - Естественный TTS-голос
  - Гибкая интеграция с аппаратным и программным обеспечением
- Цены после бесплатного: От $8/млн символов

#### 2. Baidu AI (百度)

- URL:
  - Продуктовая страница: https://ai.baidu.com/tech/mt/realtime_speech_trans
  - API/SDK для разработчиков: https://ext.dcloud.net.cn/plugin?id=22341
- Тип: Real-time speech translation API с опциональным TTS
- Задержка: Низкая задержка (毫秒级)
- Поддержка русского: ✅ Да (в списке 45 языков)
- Бесплатно: ✅ 100 часов бесплатно для тестирования
- Языки: 45 языков (включая русский)
- Особенности:
  - WebSocket протокол для потоковой передачи
  - Интеллектуальное разбиение на предложения с пунктуацией
  - Возврат TTS аудио (isReturnTts: true)
  - Оптимизировано для трансляций и конференций
- Цены после бесплатного: 10 元/час (~$1.4/час) или пакетные тарифы

#### 3. iFLYTEK (科大讯飞)

- URL:
  - Страница решения: https://global.xfyun.cn/solutions/rtime-audiotrans
  - Основная платформа: https://global.xfyun.cn/
- Тип: Real-time Audio Translation Solution
- Задержка: Низкая задержка, оптимизировано для китайского
- Поддержка русского: ✅ Да (подтверждается в описании)
- Бесплатно: ✅ Free package в консоли (требует регистрации)
- Языки: 10 языков + китайские диалекты (включая русский)
- Особенности:
  - Более 80% экономии затрат на персонал
  - Специализированные лексиконы для финансов, медицины, права
  - Аппаратные решения (переводчики, гарнитуры)
  - Опыт использования на крупных мероприятиях (China International Trade Fair)
- Цены: По запросу (enterprise-ориентировано)

#### 4. Alibaba Cloud (阿里巴巴)

- URL:
  - Онлайн переводчик: https://translate.alibaba.com
  - Продуктовая страница: https://www.alibabacloud.com/product/machine-translation
  - Qwen3 LiveTranslate: https://www.alibabacloud.com/help/en/model-studio/qwen3-livetranslate-flash-realtime

- Задержка: 100 мс на один запрос
- Поддержка русского: ✅ Да (в e-commerce и универсальной версиях)
- Бесплатно: ✅ Free quota для новых пользователей Alibaba Cloud
- Языки: 119 языков в новой версии (Tongyi Qwen)
- Особенности:
  - Две редакции: универсальная и e-commerce

  - Интеграция с Alibaba Cloud ecosystem
  - Qwen-Audio модель для multi-task audio понимания
- Цены: Pay-as-you-go после исчерпания free quota
- Важно: Для получения полного speech-to-speech опыта требуется ручная интеграция с TTS-сервисом

- Задержка: <0.5 секунд
- Поддержка русского: ✅ Да (в списке 60+ языков)
- Бесплатно: ✅ 40 минут бесплатного использования

  - AI-сводки встреч
  - Работает с Zoom, Teams, Google Meet
  - Кроссплатформенность (Web, Windows, Mac, iOS, Android)

### Рекомендации по выбору сервиса

Для лучшей поддержки русского + китайского: Tencent Cloud или Baidu AI — самые надежные варианты с прямой поддержкой русского в native API.

Для максимально бесплатного теста: Baidu AI предлагает 100 часов тестирования, Tencent Cloud — 10,000 символов/месяц постоянно.

Для B2B и enterprise: iFLYTEK — лидер с аппаратными решениями и специализированными лексиконами.

Для гибкости и масштабирования: Alibaba Cloud — если вы уже в их экосистеме и готовы к интеграции.

Для простоты использования без китайской регистрации: Transync AI — работает глобально, хотя и не является чисто китайским сервисом.

### Важные замечания

1. Регистрация: Для китайских сервисов (Baidu, iFLYTEK, Alibaba) часто требуется китайский номер телефона или WeChat для полной регистрации.
2. API vs Готовый продукт: Tencent, Baidu и Alibaba предоставляют в основном API для разработчиков, тогда как iFLYTEK и Transync AI имеют готовые UI-решения.
3. Speech-to-speech: Alibaba Cloud предоставляет только text translation, для speech-to-speech нужно комбинировать с их TTS-сервисом.

_Информация актуальна на основе публичных источников от поставщиков. Для enterprise-решений рекомендуется напрямую связаться с поставщиками._
