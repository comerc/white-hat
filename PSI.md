Ищем разработчика, которому искренне интересна тема голосовых ИИ для создания платформы цифровых психологов. Ниша стремительно растет: услуги хороших психологов стоят 5000+ рублей за сессию, их время ограничено, а спрос значительно превышает предложение. Мы создаем доступный и масштабируемый формат психологической поддержки, не привязанный к физическому времени специалиста.

## О проекте

Платформа позволяет психологам загружать свои методики, инструкции, материалы и уникальный подход. На основе этих данных создается цифровой двойник психолога. Он работает по методологии конкретного специалиста и ведет голосовые сессии с клиентами.

У нас уже есть первый психолог, который полностью подготовил инструкцию и базу знаний для первого AI-психолога.

## Дорожная карта разработки

Прототип (первая версия)

Цель: быстро получить рабочий голосовой сценарий взаимодействия клиента с AI-психологом.

- Backend на Python:
  - использование инструкций и базы знаний первого психолога
  - интеграция с GPT/LLM API
  - базовая логика голосового потокового общения S2S (Speech-to-Speech)
- Frontend (Vue или React):
  - простой интерфейс без тяжелого UI
  - одна кнопка: «Начать сессию»
- Минимальная инфраструктура:
  - логи, обработка базовых ошибок

MVP (функциональность для первых реальных пользователей)

- Аккаунты пользователей
- Кабинет психолога:
  - загрузка методик и материалов
  - управление инструкциями и настройками AI-психолога
- Биллинг:
  - подписки или платные сессии
  - интеграция с платежным провайдером
- Улучшенный, но легкий интерфейс
- Минимальная аналитика и стабильность под нагрузку
